Tablor Core - Simple UI

You can create beautiful datatables, grid views, and more using Tablor-Core

Features are already implemented, You just need your UI to work with them.
Major Features:
  • Sorting
  • Custom Sorting Toggle Orders
  • Multi-Fields Sorting
    • Example: Sorting Transactions By Bank Names, then Nested Sort By Transaction Date
  • Search
  • Search Within Previous Search Results
  • Search and merge results into previous search results
  • Different Search Criteria for Different Fields
  • Search By String Query
    • Include Specific Fields
    • Custom Fns for Converting Numbers, Dates, Booleans, Nulls, and Undefineds to Strings for Searching)
    • Search Words Using StartsWith, EndsWith, Contains, and ExactMatch Criteria
    • Search Words in Order
    • Search Words in Any Order
    • Search Consecutive Words
    • Search Non-Consecutive Words
  • Search By Date Ranges (Multiple Ranges for Multiple Fields)
  • Search By Number Ranges (Multiple Ranges for Multiple Fields)
  • Search By Exact Values (Multiple Exact Values for Multiple Fields)
  • Search By Custom Search Fn
  • Reverting Search Results (replace searched items with those that were not searched)
  • Pagination
  • Go To Page
  • Set Number of Items Per Page
  • All Items in Single Page
  • Select Items
  • Get Selected Items
  • Get Selected Items that are in the Page
  • Get Selected Items that are not in the Page
  • Completely Event Based
    • Example: Events for Page Change, Sorting Change, Search Change, Item Selection Change, Deletion and more.
